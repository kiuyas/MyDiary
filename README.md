# MyDiary
## My Simple Diary

- シンプルで広告がない日記アプリがほしくて自作を開始しました。
- まだ作りかけです。
- ひとまず以下の機能があります。
  - 月めくりができる
  - 日々の内容が入力できる
  - 入力した内容はファイルに保存される
    - 保存された内容はあとでまた表示される。
- というわけで、一応使うことができるレベルにはなっています。
- 今後、ブラッシュアップをしていって使いやすくする予定です。
  - 画像添付もできるようにしたいです。
